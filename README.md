## Описание
BookTrackerApp создано для автоматизации процесса выбора и чтения книг.  
С помощью приложения можно:
- просматривать список доступных книг;
- фильтровать их по жанрам;
- искать по названию или автору;
- добавлять книги в «Избранное» со статусами:
  - «Хочу прочитать»  
  - «В процессе»  
  - «Прочитано»;
- ставить личные оценки книгам.
